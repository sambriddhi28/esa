# Extended Stay America Cincinnati - Guest Welcome Page

A professional, interactive welcome guide for guests at Extended Stay America Cincinnati-Covington property.

## 📖 Features

✨ **Beautiful Design**
- Modern, responsive interface
- Works perfectly on desktop, tablet, and mobile
- Brand-aligned color scheme

📸 **Image Carousel**
- 3 hotel property photos
- Auto-rotate or manual navigation
- Smooth transitions

📺 **Comprehensive Information**
- Check-out times and breakfast hours
- Fitness room access times
- Laundry and vending machine locations
- Pet policies
- Available amenities

📡 **TV Navigation Guide**
- Visual guides for TV remote controls
- Helps guests find channels easily

🎬 **Complete Channel Listing**
- 150+ TV channels organized
- Searchable channel database
- Channel codes and numbers
- Real-time search filtering

📱 **Mobile Optimized**
- Perfect for guest phones
- Easy to navigate on small screens
- Shareable via link or QR code

## 📱 Creating a QR Code

Once deployed, generate a QR code using:
- QR Code Monkey: https://www.qrcode-monkey.com/
- Google Chart API
- Any free QR code generator

Print the QR code and place in:
- Guest welcome packets
- Hotel room TVs
- Lobby notice boards
- Check-in desk
- Elevators

Guests simply scan with their phone camera!

## 🔧 How to Use

### As a Guest:
1. Scan QR code OR visit the link
2. View hotel information
3. Search for TV channels
4. Access amenities guide
5. Contact front desk with questions

### As an Administrator:
1. Edit `index.html` with any text editor
2. Update hotel information as needed
3. Add/remove channels from the list
4. Update images
5. Re-deploy to see changes
